📱 Overview

Tribe Up is a modern, scalable Social Media mobile application built using Flutter and structured with Clean Architecture principles.

The goal of this project is to simulate a real-world production-ready social media app with proper separation of concerns, modular feature-based structure, and maintainable codebase.

🏗 Architecture

This project follows Clean Architecture with feature-first structure:

lib/
│
├── core/
│   ├── network/
│   ├── utils/
│   └── constants/
│
├── features/
│   ├── auth/
│   ├── home/
│   ├── profile/
│   ├── posts/
│   ├── comments/
│   └── chat/
